# chat-app-server


## :tv: See it live

[https://simple-react-chat-appl.netlify.app/](https://simple-react-chat-appl.netlify.app/)

## :computer: General info

A chat app made with Socket.io and Node. ( :

## :cake: Features

- Separated chats by room.
- Warning when someone enter or exit a room.
- List of room's users.

## :rocket: Technologies

Project created with:

- [Socket.io](https://socket.io/)
- [Node](https://nodejs.org/)
- [Express](https://expressjs.com/)

## :triangular_ruler: Setup

To see it working you have to clone and run the front end app after run the server [Chat-app](https://github.com/henrique-lothammer/chat-app).

To run the server, install it locally using yarn:

```
# Install dependencies
$ yarn
```

And finally, run it:

```
# Run the server
$ yarn start

```
